About Ornament Generation

This demo uses a PerTok-based Transformer model trained with a specific ornament-aware loss function, capable of adding natural, stylistically appropriate ornaments to MIDI files.

Model Features:

  • PerTok tokenization supporting microtiming representation
  • Domain-specific Transformer architecture
  • Ornament-aware loss function
  • Trained on 530 classical pieces from the MAESTRO dataset
